EBCDIC to ASCII Converter

The EBCDIC-ASCII converter allows real-time conversion between EBCDIC (Extended Binary Coded Decimal Interchange Code) and ASCII (American Standard Code for Information Interchange) character encodings. The tool features a split-screen interface with two text areas - one for ASCII input and another for EBCDIC output. The converter includes comprehensive character mapping for common characters, including letters, numbers, special characters, and control characters. This tool is particularly useful for mainframe developers and system integrators who frequently work with both encoding standards.

Indian Stock Market Indices Tracker

A dynamic real-time tracker for major Indian stock market indices including NIFTY 50, BSE SENSEX, NIFTY Bank, NIFTY IT, NIFTY Auto, and NIFTY FMCG. The tool features an interactive line chart showing 30-day performance history and a detailed data table displaying key metrics like open, high, low, close prices, and daily changes. Users can switch between different indices using a dropdown selector, with data automatically refreshing every 5 minutes. The tracker uses color-coded indicators (green for positive, red for negative changes) to provide quick visual feedback on market performance. This tool is particularly useful for investors and traders who need to monitor Indian market indices throughout the trading day.
